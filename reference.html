Java Script Reference

This section refers to key JavaScript things to remember.

Javascript can be included in a HTML webpage either:

  • Including <script>...code here...</script> in the <head> part of the html file.
  • Including <script>...code here...</script> in the <body> part of the html file.
  • Or by referring to a separate .js file by including <script src="script.js"></script> in the body; note that there is nothing between the tags in this instance.
Comments & Variables: 
// Comments in Javascript are made with two backslashes and only apply to this line.
// Each new line of comment needs a new pair of backslashes.
let counter = 0;
counter = counter + 1;
counter += 1;
counter++;
Conditionals: 
if (x < y)
{
    ...
}
else if (x > y)
{
    ...
}
else
{
    ...
}
Loops: 
while (true)
{
    ...
}

for (let i = 0; i < 3; i++)
{
    ...
}
To define a function in Javascript: 
function greet()
{
    let name = document.querySelector('#name').value;
    alert('hello, ' + name);
}

The above would take the variable "name" and display this as part of a pop-up alert.

You can use Event Listeners to "listen" for user actions. For example:
function listen()
{
    document.querySelector('form').addEventListener('submit', greet);
}
    document.addEventListener('DOMContentLoaded', listen);

The above will wait until the page is loaded, then take input from the form on the page and then runs a function called greet.

You can also use anonymous functions (when a function is only going to be used once) as follows within the script/head: 
document.addEventListener('DOMContentLoaded', function() {
    document.querySelector('form').addEventListener('submit', function () {
            let name = document.querySelector('#name').value;
            alert('hello, ' + name);
        });
    });

The above is calling an anonymous or Lambda function, that includes two functions (which were previously seperated out into Listen and greet).

All kinds of events can be listened for, including (but not limited to!) things like click, drag, focus, keyup, mousedown, mouseover etc.

HTML symbols etc

While typing in HTML you may intend an & or < for example to appear in the text, but as these elements have "code" meanings in HTML you cannot simply type these symbols.

For an & to appear, you must type &amp;. For > to appear you must type &gt;. For < to appear you must type &lt;.

You must remember to terminate these items with a semicolon ';'.

HTML Paragraphs and breaks

To move to new line within the same paragraph or section, use <br />. Although using <br> and </br> on their own also seem to do the same thing. According to this article using <br> is perfectly fine in HTML, however the / variants should be used for XHTML. So you should use <br/> for flexibility.

To add in a line break, with a whole clear line, use: <br></br>

To add extra spaces (as HTML will ignore multiple spaces in text), you can use &nbsp or you can use &#160 instead. Two spaces can be inserted with &ensp; and four spaces inserted with &emsp;.

HTML Forms

Forms can be created by using <form>...</form> followed by details of the form itself, such as:

<form>
    <input id="name" placeholder="What's your name?" type="text">
    <input type="submit" value="Say hello to me!">
</form>
The above code will produce the below form (which is a dummy only):

We can use the form to interact with Client side Javascript functions, for example:

<form onsubmit="greet(); return false;">
    <input id="name" type="text">
    <input type="submit">
</form>

The above would need a javascript function to be defined; but would run the function then do nothing else (return false).

HTML Tools

You can make a page automatically refresh at a defined time interval using the below code:

<meta http-equiv="refresh" content="2">

Where in the above example the page will refresh every 2 seconds.


CSS Reference

This section refers to key CSS things to remember.

Use the following code within the 'head' of the webpage to take formatting from a separate CSS file instead of simply from within the HTML file. <link href="styles.css" rel="stylesheet">

Example of text as a Tag styled from the styleheet. HTML used: <code>...</code>. Included in the stylesheet as simply:

code
{
    color:....;
}

Example of text as a Class styled from the stylesheet. HTML used: <class="exampleClass">. Included in the stylesheet as 

.exampleClass
{
    color: ....;
    background-color: ....;
}

You can use multiple classes grouped together.

Example of text as a ID styled from the stylesheet. HTML used: <id="exampleID">. Included in the stylesheet as 

#exampleID
{
    color: ....;
    background-color: ....;
}
